# json2script
parse json to script tools / 将json转换成script的工具

正在抓紧开发和完善中...

## installation
json2script is a registered npm module. So you can install it using the following command:
```
npm install json2script
```

推荐使用gulp 请参照本项目中的gulpfile.js的编写
